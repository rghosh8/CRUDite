Code Ninja is an interactive Javascript IDE for Chrome. It saves your code to your PC's local storage and can retrieve saved code long after submission. Write, edit, update and delete code with the click of a button.

Your code will never break formatting and will last as long as your computer does. After running your code, press Command + Option + J to see the result.
